Инструменты для экономического анализа вмешательств по ведению пациентов с сердечной недостаточностью Модель экономической эффективности: веб-программа, предназначенная для оценки экономической эффективности программ ведения пациентов с сердечной недостаточностью.
фон:
Программы лечения заболеваний с сердечной недостаточностью могут влиять на использование медицинских ресурсов и выживаемость с поправкой на качество. Поскольку прогнозирование долгосрочных затрат и выживания является сложной задачей, последовательный и обоснованный подход к экстраполяции краткосрочных результатов был бы ценным.
методы:
Мы разработали инструменты для экономического анализа вмешательств по ведению пациентов в рамках модели экономической эффективности сердечной недостаточности, веб-инструмента моделирования, предназначенного для интеграции данных о демографических, клинических и лабораторных характеристиках; использовании лекарств, основанных на фактических данных; и затратах для получения прогнозируемых результатов. Прогнозы выживания основаны на модифицированной модели сердечной недостаточности в Сиэтле. Прогнозы использования ресурсов и качества жизни моделируются с использованием взаимосвязей с изменяющимися во времени показателями модели сердечной недостаточности в Сиэтле. Модель может быть использована для оценки проектов параллельных групп и одиночных когортных исследований и гипотетических программ. Моделирование состоит из 10 000 пар виртуальных когорт, используемых для получения оценок использования ресурсов, затрат, выживаемости и дополнительных коэффициентов рентабельности на основе пользовательских данных.
результаты:
Модель продемонстрировала приемлемую внутреннюю и внешнюю валидность при воспроизведении оценок использования ресурсов, затрат и выживаемости по результатам 3 клинических испытаний. Моделирование для оценки экономической эффективности программ лечения сердечной недостаточности по трем сценариям демонстрирует, как модель может быть использована для разработки программы, в которой краткосрочных улучшений функционирования и использования методов лечения, основанных на фактических данных, достаточно, чтобы продемонстрировать хорошую долгосрочную ценность для системы здравоохранения.
заключение:
Инструменты для экономического анализа вмешательств по ведению пациентов при сердечной недостаточности Модель экономической эффективности предоставляет исследователям и поставщикам медицинских услуг инструмент для проведения долгосрочного анализа экономической эффективности программ лечения заболеваний при сердечной недостаточности.
